
# Sarmaya (سرمایہ) - Portfolio Manager Landing Page

Modern, responsive landing page for **Sarmaya**, a comprehensive portfolio management platform for crypto and stocks.

## 🌟 Overview

Sarmaya is your ultimate portfolio manager that tracks both cryptocurrency and stock investments with accurate FIFO calculations, multi-currency support (EUR/USD), and privacy-first design.

**Live Demo:** [sarmaya.app](https://sarmaya.app)
